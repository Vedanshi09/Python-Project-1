Instantiation

Instantiation is another word for create. In OOPs the term instantiation is to create a class or object.

Object

OOPs has a term called “Object” which consists of different methods and properties. This helps to create specific types of data which are useful.

Class

A class is like a template in which we can make objects in order to execute a different program.

Namespace

Namespace was created to make programming language organized. Every namespace has a specific meaning.

Constructor

The constructor does a unique work in OOPs. Its purpose is to initialize an object of a class in order to execute a code

Fixture

Fixture is another word of sample data. It is a set of objects that is used as a baseline to run tests.

Type hint

Type hints basically provide hints that can force function to accept desired data type.

Type cast

The function performed through type cast is that we are able to change a way of an object from one data type to another.

Unit test

The purpose of performing a unit test is to test the correctness of isolated code and clear out if there are any errors.
