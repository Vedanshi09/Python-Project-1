Inheritance

Inheritance is a method in OOP where we can use the mechanism of the parent class in another class. We do that by specifying implementation to maintain the same behavior of class.

Polymorphism

Polymorphism is a concept in OOPS which describes the ability of a message that can be displayed in more than one form. That is, it is a situation that occurs in several different forms.

Abstraction

An abstract method is simply a method without an implementation. In this method a programmer can hide all except the relevant data in order to reduce complexity and increase efficiency.
